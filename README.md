# chord_tracking

## 基于 AutoChord 项目

本頁面基於 Autochord團隊所撰寫之UI介面進行拓展,功能上基本上一樣(顯示和弦進行),只是多了兩個軌道可以做比較 [AutoChord](https://github.com/cjbayron/autochord) 。感谢 CJ Bayron 及其他開發者的開源項目．
